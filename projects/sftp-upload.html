Goal

Brackets is a full-fledged code editor for web development maintained by Adobe. I wrote this plugin to allow web developers upload their code to their server through FTP or SFTP protocols.

Description

brackets-sftp-upload is designed for field needs for web developers: it allow tracking changed files in a project; uses can select which file to upload and which one to skip; it can save server settings for each project.
