Reduce document circulation cycle of a state-owned company from 30 days to 3 days (Business project).

Goal

design and develop a office automation (OA) system for document circulation, process approval, and document sharing in a investment company.

Description

This system has a document circulation module, a business workflow module (implemented by JBPM), a file sharing module, and a flexible role-based authentication module. It has a Adobe Flex front-end, so it can achieve a decent speed at that time and suitable to multiple browsers; it has a java back-end and uses MS SQL Server for data storage. It is a summer project for a company, we have 4 team members.
