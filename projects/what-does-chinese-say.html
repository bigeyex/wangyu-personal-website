Search and read Chinese news and social media without knowing Chinese (Hobby project).

Links

Demo Site Github

Goal

allow non-Chinese speaking people search for Chinese news and social media content in a single web interface

Description

searches are first translated from English to Chinese using Baidu translate API, then searches are performed via Baidu News Search; the result is translated back to English for reading.
The version in Github uses web sockets, so results are shown up one by one. The server which hosts the demo site does not support web sockets, so the speed of the demo site might be a bit of slow.
