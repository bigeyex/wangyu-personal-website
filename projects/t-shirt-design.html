A playable memento for a graduating class.

Goal

Create a T-shirt design that help remembering members of the class and memories spent together with the class.

Description

The back of this t-shirt is a playable chess board. Every slot in this chess board is made from a picture of a classmate, and stepping into each slot will "trigger events" specified with that person's personality in mind.
