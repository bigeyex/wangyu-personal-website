Goal

Although hackathons are popular today, most of their influence is constrained within no more than 48 hours; and participants in hackathons often repeat works done in other eventshappened before. Civic Coding Collector (CCC) a tool designed for creating long-term engagement in civic coding activities. It encourage cross-sector (coder, designer, activist; new-comers, veterans) participation around civic coding topics.

Description

CCC allow the user to collect Github repositories(repos) related to civic coding. After installing a Google Chrome extension, users can bookmark Github repos in the reposâ€™ project pages by clicking a button. After that, users may add descriptions, add the repo to one of the lists, and discuss with other users.
This project also includes a web backend, allowing users to create lists, reply in discussions, and submit ideas related to civic coding.
This project is still at prototyping stage.
