Goal

Create an arcade game for MIT museum, taking "music" as a key element.

Description

This competitive game lets players dodging bullets while pushing buttons with the beat of the music. Great performance create additional bullets to your opponent's space. This game is designed for MIT museum (whose visitors may be groups of small children), so we delivered a short, intense, and fun experience (for visitors to take turns).
